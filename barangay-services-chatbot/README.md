# Barangay Services Chatbot

Beginner project for building a small public-service chatbot with HTML, Express, and Aldabra AI Gateway.

## What you build

A one-page chatbot that can answer common questions about:

- barangay ID requirements
- permit steps
- office hours
- contact information

## Run locally

```bash
npm install
cp .env.example .env
npm run dev
```

Open:

```text
http://localhost:3000
```

## Why this is beginner-friendly

- The API key stays in the backend.
- The frontend is a plain HTML file.
- The model name is stable: `fast-chat`.
