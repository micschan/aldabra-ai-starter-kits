# Multi-Model Router Demo

Compare several low-cost OpenRouter models through one Aldabra endpoint.

## Models

```text
deepseek/deepseek-chat-v3-0324
qwen/qwen-2.5-72b-instruct
google/gemini-2.5-flash-lite
deepseek/deepseek-r1-distill-qwen-32b
```

## Run locally

```bash
npm install
cp .env.example .env
npm run dev
```
