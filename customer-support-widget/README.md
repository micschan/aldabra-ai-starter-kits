# Customer Support Widget

Beginner-friendly support chat widget for product FAQs, delivery updates, and account questions.

## Model

Uses the specific low-cost OpenRouter model:

```text
deepseek/deepseek-chat-v3-0324
```

## Run locally

```bash
npm install
cp .env.example .env
npm run dev
```

Open:

```text
http://localhost:3000
```
